# Contributing to ctxqa

Thank you for your interest in contributing to ctxqa!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<username>/ctxqa.git`
3. Create a feature branch: `git checkout -b feat/your-feature`
4. Make your changes
5. Run tests: `go test ./...`
6. Commit your changes: `git commit -m "feat: add your feature"`
7. Push to your fork: `git push origin feat/your-feature`
8. Open a Pull Request

## Code Style

- Follow Go best practices and idioms
- Use `gofmt` or `goimports` for formatting
- Write meaningful commit messages following Conventional Commits
- Add tests for new functionality

## Report a Bug

Open an issue at https://github.com/ctxqa/ctxqa/issues with:
- A clear description of the bug
- Steps to reproduce
- Expected vs actual behavior
- Your environment (OS, Go version)
